# QA Automation Engineer (Cypress) Task for 10Clouds

## Getting Started

**Pre-requisite**
[node-must-be-installed](https://nodejs.org/)

```
git clone https://github.com/Jerry1811/qa-task-10clouds
cd qa-task-10clouds
```

**Install project dependencies**

```
npm install
```

## Running Test

```
npm test
```

**Run tests in headless mode**

```
npm run cy:run
```
